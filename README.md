# polor-pashm
polor is simple **Terminal Color** library for _Pashmak_ programming language.

# preview
```html
import 'polor.pashm'

use polor

println $FgBlue + 'Pashmak'
println $BgHiGreen + $FgHiBlack + 'programming' + $Reset
println $Bold + 'language'
```
